Οδηγίες εγγραφής στη πλατφόρμα

Η εγγραφή στη πλατφόρμα είναι πολύ απλή! Ο χρήστης θα πρέπει να πατήσει το κουμπί «Είσοδος / Εγγραφή» που βρίσκεται στο κάτω μέρος αυτής της σελίδας αλλά και στο επάνω δεξιά μέρος του site και να συμπληρώσει τα πεδία που θα του ζητηθούν. Αφού βεβαιωθεί ότι έχει πληκτρολογήσει τα σωστά στοιχεία θα πρέπει να πατήσει το κουμπί «Εγγραφή» που βρίσκεται ακριβώς κάτω από τα πεδία που θα συμπληρώσει. Ιδιαίτερη προσοχή θα πρέπει να δοθεί στα πεδία που είναι υποχρεωτικά, καθώς η συμπλήρωσή τους είναι απολύτως απαραίτητη για την επιτυχή ολοκλήρωση της διαδικασίας εγγραφής!

Τα πεδία που ο χρήστης καλείται να συμπληρώσει κατά τη διαδικασία εγγραφής είναι:

  • Όνομα (Υποχρεωτικό)
  • Επώνυμο (Υποχρεωτικό)
  • Α.Μ.Κ.Α. (Υποχρεωτικό)
  • Α.Φ.Μ. (Υποχρεωτικό)
  • Αριθμός ταυτότητας (Υποχρεωτικό)
  • Ηλικία (Υποχρεωτικό)
  • Φύλλο
  • Email
  • Κινητό τηλέφωνο (Υποχρεωτικό)

Οδηγίες εισόδου στη πλατφόρμα

Για τη πραγματοποίηση εισόδου στο σύστημα, ο χρήστης θα πρέπει να πατήσει το κουμπί «Είσοδος / Εγγραφή» που βρίσκεται στο κάτω μέρος αυτής της σελίδας αλλά και στο επάνω δεξιά μέρος του site και να συμπληρώσει τα πεδία που θα του ζητηθούν. Αφού βεβαιωθεί ότι έχει πληκτρολογήσει τα σωστά στοιχεία θα πρέπει να πατήσει το κουμπί «Είσοδος» που βρίσκεται ακριβώς κάτω από τα πεδία που θα συμπληρώσει. Για να μπορέσει ένας χρήστης να χρησιμοποιήσει τη πλατφόρμα θα πρέπει πρώτα να εγγραφεί σε αυτή με τη διαδικασία που αναφέρεται παραπάνω!

Τα πεδία που ο χρήστης καλείται να συμπληρώσει κατά τη διαδικασία εισόδου είναι:

  • Α.Μ.Κ.Α. (Υποχρεωτικό)
  • Α.Φ.Μ. (Υποχρεωτικό)
